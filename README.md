# Webpage_Andres
Repositorio para trabajar en conjunto y desarrollar la posible página para el papá de Andrés.


## Rama de ideas

En esta rama se propondran las ideas o posibles diseños para la página web.

Pueden proponer cambios al diseño de la página, proponer funcionalidades nuevas, reeestructurar la página entre otos.

**Recuerden primero subirlo a esta rama y no a la rama principal (main)** ya que primero hay que revisar cada uno de los cambios y que todos estemos de acuerdo  con cada uno de los cambios propuestos.

También podemos proponer otras maneras de organizar el proyecto, ya sea agregando mas ramas o diferentes cosas.
